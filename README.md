# WEB-BankApp

Banka Uygulaması 

Bulundurduğu Özellikler:

-Kullanıcı Kayıt
-Kullanıcı Giriş 
-Hesap Oluşturma
-Hesap Kapatma
-Hesap Listeleme
-Bakiye Yükleme
-Bakiye Çekme
-Havale/ Virman İşlemleri
-Hgs'ye Sistem Üzerinden Para Yükleme
-Kullanıcının Kredi Çekip Çekemeyecegini Hesaplayan Kısım (EKLENECEK)
